1. What is different between local storage and sesson storage?

Ans:There are some different between local storage and sesson storage.Such as:the local storage data does not expire anytime but the session storage data is cleared when the page session ends,the local storage data is opened in a new tab,or the browser is closed and reopened but the session storage data creat a new session and new storage for us when we open a new tab or a new window,local storage works on same-origin policy.so,data stored will only be available on the same origin but the session storage data is availavail only inside the window/tab in which it was set,local storage is not session based so if we want deleate it we have need JS or manually client side reading only but session storage is session based and it work per window / tab client side reading only.
